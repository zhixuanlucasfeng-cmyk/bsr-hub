# Live Demo Script

1. Open the homepage and show the three choices: rent, workspaces, and second-hand.
2. Search `PS5`, open **PS5 Slim + Two Controllers**, choose local delivery and two 30-minute units.
3. Click **Get Rust price** and point out base price, 6% service fee, delivery, deposit, and BSR-held total.
4. Reserve the item. In **My orders**, click **Complete protected payment** as Maya.
5. Switch the top-right persona to Jordan and click **Seller confirms**, then **Start rental**.
6. Switch back to Maya and click **Return item**.
7. Switch to Jordan, click **Complete order**, then show the review button.
8. If the API is unavailable, run `npm run demo`, reload, and repeat. If time is short, show the completed-order screenshot and explain that every action is enforced by Rust.
